Application Name: Chat On Web
Developed By: Yash Gandhi
Demo Link: https://chatonweb.yashgandhi.ca/

Brief Description:
Chat on Web is XML, PHP and Javascript based chat application. This web application was developed to perform XML and PHP fundamentals. This web application can be used by multiple users parallely. The users can sign in with the help of developed login functionality or using Google authentication. 

Steps to execute:
1) Start local server example- Apache
2) Go to browser
3) Run URL as localhost://FOLDER NAME/
Note: FOLDER NAME is the project folder where the files are downloaded.
4) Register as new user or login using Google account.
